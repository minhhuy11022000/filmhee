# Filmhee API

Backend service — **coming soon**.

## Planned tech stack

- **Framework**: Nest.js or Node.js + Hono (TBD)
- **Language**: TypeScript
- **Database**: TBD
- **Auth**: TBD

## Planned endpoints

- `GET /films` — list films with pagination and filters
- `GET /films/:id` — film detail
- `GET /users/:id` — user profile
- `PATCH /users/:id/watchlist` — add/remove from watchlist
- `GET /users/:id/history` — viewing history
- `POST /auth/login` — authentication
- `POST /auth/register` — registration
